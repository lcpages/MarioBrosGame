# initial-project
---------------

Little game developed with C-library **SDL**

Objectives: Move boxes to green points 

Keyboards : up, down, left, right 
          -Map creation mode : 
            4 : Wall
            3 : Box
            2 : Mario
            1 : Objectif

  Rest of implementation :
    -Score counter 
    -Real horizontal Mario Bros Game mode
    -Selection of personnalized maps
